# Placeholder Images

Default placeholder images for products and marketing sections.

## Files

- `product-1.jpg` - Product placeholder (watch on white)
- `product-2.jpg` - Product placeholder (white sneaker)
- `product-3.jpg` - Product placeholder (clothing item)
- `product-4.jpg` - Product placeholder (fashion items)
- `hero.jpg` - Hero section default image
- `lifestyle.jpg` - Lifestyle/editorial section image

## Usage

These images are automatically used as fallbacks when:
- Products don't have a thumbnail image
- Hero section needs a default image
- Lifestyle sections need placeholder content

## Replacement

To customize:
1. Replace these images with your brand's photos
2. Maintain similar aspect ratios:
   - Products: 3:4 portrait (800x1000px)
   - Hero: 4:5 portrait (1200x1600px)
   - Lifestyle: 4:5 portrait (1000x1400px)
